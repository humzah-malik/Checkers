Detailed Description

Represents an AI-controlled player in the Checkers game.

The AI class extends Player and uses either random move selection or the Minimax algorithm to decide its next move, depending on the selected difficulty by the user.

Determines the best move for the AI based on its difficulty level.

Uses random selection (easy) or minimax (medium/hard) depending on difficulty.

Parameters

Returns
A pair consisting of the Piece to move and its target position (row, col).
  • Easy: makes a random move.
  • Medium: uses the minimax algorithm with 200 ms time-limit.
  • Hard: uses the minimax algorithm with 350 ms time-limit.
