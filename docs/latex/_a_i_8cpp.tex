\doxysubsection{Detailed Description}
Implements the \doxylink{class_a_i}{AI} class responsible for making moves in the Checkers game. 

This class handles \doxylink{class_a_i}{AI} behavior, including selecting a move using either random selection or the Minimax algorithm, depending on the difficulty level selected by the user.
