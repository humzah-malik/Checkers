\doxysubsection{Detailed Description}
Implements the Minimax algorithm with alpha-\/beta pruning for \doxylink{class_a_i}{AI} gameplay in Checkers. 

Contains logic for recursive game state evaluation, move simulation, board evaluation heuristics, and iterative deepening with time constraints. This class is used by the \doxylink{class_a_i}{AI} player to decide optimal moves.
