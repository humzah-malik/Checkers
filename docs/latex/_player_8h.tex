\doxysubsection{Detailed Description}
Implements the \doxylink{class_player}{Player} class used to represent a human or \doxylink{class_a_i}{AI} player in Checkers. 

Provides methods for making moves, accessing player pieces, and retrieving player information. Acts as a base class for player-\/related operations in the game.
