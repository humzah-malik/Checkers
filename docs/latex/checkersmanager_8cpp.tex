\doxysubsection{Detailed Description}
Implements the \doxylink{class_checkers_manager}{Checkers\+Manager} class for managing the overall game state and control flow. 

This file manages game startup, \doxylink{class_a_i}{AI} and player interaction, turn logic, undo functionality, and win condition checks. It connects the game board to either player-\/vs-\/player or player-\/vs-\/\+AI logic depending on selected settings.
