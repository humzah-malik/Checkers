\doxysubsection{Detailed Description}
Represents an AI-\/controlled player in the Checkers game. 

The \doxylink{class_a_i}{AI} class extends \doxylink{class_player}{Player} and uses either random move selection or the Minimax algorithm to decide its next move, depending on the selected difficulty by the user.
