\doxysubsection{Detailed Description}
Implements the main in-\/game UI for a Checkers match. 

Sets up the game layout, controls (undo, back, settings, exit), and displays turn indicators, undo counts, and game-\/over dialogs. Handles user interactions and connects to \doxylink{class_checkers_board}{Checkers\+Board} events.
