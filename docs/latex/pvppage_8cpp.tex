\doxysubsection{Detailed Description}
Implements the UI for the \doxylink{class_player}{Player} vs \doxylink{class_player}{Player} selection screen in the Checkers game. 

Provides options for choosing between 1v1 and 4-\/player multiplayer modes (with the latter currently not implemented). Includes navigation buttons for going back, accessing settings, and exiting the application.
